This repository contains runscripts for the paper 

"*Call for participation: Verification benchmarks for single-phase flow in three-dimensional fractured porous media*" by Inga Berre, Wietse Boon, Bernd Flemisch, Alessio Fumagalli, Dennis     Glaser, Eirik Keilegavlen, Anna Scotti, Ivar Stefansson, Alexandru Tatomir. See [arXiv pre-print](https://arxiv.org/abs/1809.06926).

To run the file, PorePy should be installed. As the runscript in this repository have not been updated along with PorePy, you may need to revert to an older version of PorePy.

This file should work with PorePy commit 367c31373df6029d1bc5f19691f77d944e869400
